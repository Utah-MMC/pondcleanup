# Pond Authority

A nationwide contractor finder platform for pond and water-feature services.

## Project Structure

```
pondauthority/
├── index.html                    # Homepage
├── css/
│   └── styles.css               # Main stylesheet
├── pages/
│   ├── find-a-contractor.html   # Search/directory page
│   ├── services.html            # Services overview
│   ├── how-it-works.html        # How it works page
│   ├── about.html               # About page
│   ├── contact.html              # Contact page
│   ├── for-contractors.html     # Contractor signup page
│   └── contractor/
│       └── bluewater-pond-garden.html  # Sample contractor profile
└── README.md
```

## Features

- **Homepage** with hero search, how-it-works, popular services, and featured contractors
- **Search/Directory** page with filters and contractor listings
- **Contractor Profiles** with overview, services, gallery, reviews, and quote form
- **Additional Pages** for services, how-it-works, about, contact, and contractor signup
- **Responsive Design** - Mobile-first approach with breakpoints for tablet and desktop
- **Modern UI** - Water/nature color palette with clean typography

## Design System

### Colors
- Primary: Deep blue (#0066cc)
- Secondary: Teal (#00a3a3)
- Accent: Soft green (#2d8659)
- Background: White with light gray sections

### Typography
- Font: Inter (with system font fallbacks)
- Clean, readable sans-serif throughout

## Getting Started

Simply open `index.html` in a web browser to view the homepage. All pages are static HTML with CSS styling.
